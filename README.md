# 맛Spot 🍽️ - 나만의 맛집 지도

## 실행 방법

1. index.html을 VS Code에서 열고 Live Server로 실행 (권장)
   또는 브라우저에서 직접 더블클릭

## 카카오맵 API 키 발급 (무료, 5분)

1. https://developers.kakao.com 접속 후 로그인
2. 내 애플리케이션 → 애플리케이션 추가하기
3. 앱 이름 "맛Spot" 입력 후 생성
4. 앱 키 탭에서 **'JavaScript 키'** 복사
5. 앱 설정 → 플랫폼 → Web → 사이트 도메인 등록
   - 로컬: `http://localhost:5500` 또는 `http://127.0.0.1:5500`
6. index.html 상단 스크립트의 `YOUR_KAKAO_KEY` 부분에 키 붙여넣기

## 주요 기능

- 🗺️ 카카오맵 기반 실시간 맛집 지도
- 📊 네이버/구글/블로그/인스타 통합 평점
- ⚡ 실시간 대기 현황 + 시간대별 혼잡도
- ⭐ 나만의 맛집 등록/수정/삭제 (localStorage)
- 🔍 이름/카테고리/태그 실시간 검색
